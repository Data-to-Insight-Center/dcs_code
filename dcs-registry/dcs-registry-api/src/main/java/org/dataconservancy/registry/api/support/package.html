Provides supporting classes of the Data Conservancy Registry Framework API

Summary

These interfaces are likely to be used by {@link org.dataconservancy.registry.api.Registry Registry} implementations. The model mapper interfaces provide mechanisms for converting between a domain object and packaging formats (currently the Data Conservancy Packaging is supported). The {@link org.dataconservancy.registry.api.support.RegistryEntry} is an abstraction over domain objects, allowing model mapping interfaces to work generically with domain objects.
